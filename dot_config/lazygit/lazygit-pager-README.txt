lazygit-pager.bat について
===========================

このファイルは BAT と JScript のハイブリッドスクリプトです。

【概要】
- @if(0)==(0) は BAT では true、JScript では false になる条件式を使用して、
  両言語を条件分岐で共存させています

【BAT 部分の処理】
1. コードページ設定：UTF-8 (65001) に設定
2. パラメータ取得：
   - old: 前のファイルパス（2番目の引数）
   - new: 新しいファイルパス（5番目の引数）
   - target: ターゲットパス（1番目の引数）
3. パス区切り文字の変換：バックスラッシュ → スラッシュ
4. パイプ処理の流れ：
   (1) git で差分を出力
   (2) 最初の cscript: 古いテンポラリファイルパスをファイル名に置換
   (3) 2番目の cscript: 新しいテンポラリファイルパスをファイル名に置換
   (4) delta でシンタックスハイライト付き差分を表示

【JScript 部分の処理】
- @end 以下が JScript として実行される
- コマンドライン引数から置換対象文字列と置換文字列を取得
- 標準入力からパイプされた行を1行ずつ読み取り
- 指定された文字列を置換して標準出力へ出力（パイプラインの次のプロセスへ）
