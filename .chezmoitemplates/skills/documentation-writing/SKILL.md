# ドキュメント作成スキル

## ワークフロー

1. 対象を確認する（何を書くか・誰向けか・新規か更新か）
2. 更新の場合は既存ドキュメントを読む
3. 必要な情報を収集する（実装コード、仕様など）
4. 執筆する
5. チェックリストで検証する

## チェックリスト

ドキュメント作成・更新時に確認：

- [ ] 冗長性を排除し、読み手に必要な情報のみ含まれているか
- [ ] すべての文が能動態か、受動態の場合は必要か
- [ ] 記載内容に偽りや矛盾がないか
- [ ] リンクや参照先は存在するか
- [ ] 用語が統一しているか
