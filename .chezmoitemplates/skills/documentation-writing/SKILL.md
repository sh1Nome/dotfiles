# ドキュメント作成スキル

## 基本方針

ドキュメント作成時は以下の原則に従う：

1. 冗長性を避け、必要な情報に絞り記載する
2. なるべく受動態を使わない
3. 記載内容に偽りや矛盾がないか確認する

## 実行チェックリスト

ドキュメント作成・更新時に確認：

- [ ] 読み手にとって必要な情報のみ含まれているか
- [ ] すべての文が能動態か、受動態の場合は必要か
- [ ] 記載内容が実装コードと一致しているか
- [ ] リンクや参照先は存在するか
- [ ] 用語が統一しているか
